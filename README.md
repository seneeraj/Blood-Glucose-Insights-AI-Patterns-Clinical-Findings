# 🧠 AI Glucose Pattern Analyzer

An AI-powered application that analyzes blood glucose logs to detect **hidden clinical patterns** that may be missed during routine review.

The system supports **Excel, CSV, and Image-based glucose logs**, extracts glucose readings, detects patterns, and presents insights through an interactive dashboard.

---

# 🚀 Features

✔ Upload glucose logs (Excel, CSV, Image)  
✔ Automatic data extraction and cleaning  
✔ Detection of clinical glucose patterns  
✔ Glucose trend visualization  
✔ Glycemic heatmap  
✔ Meal response analysis  
✔ Clinical summary interpretation  
✔ Pattern ranking engine  
✔ Doctor-friendly dashboard

---

# 📊 Example Insights

The system can detect patterns such as:

- Severe hyperglycemia
- Hypoglycemia risk
- Post-meal glucose spikes
- Glycemic variability
- Possible insulin resistance
- Dawn phenomenon
- Basal insulin mismatch

# 📈 Dashboard Components

The dashboard provides:

### Glucose Overview
Summary statistics including:

- Average glucose
- Variability
- Maximum / Minimum values

### Top Clinical Patterns
Highlights the most important detected patterns.

### Glucose Trend Chart
Shows daily glucose trends across meal readings.

### Glycemic Heatmap
Visualizes glucose risk zones across days and meals.

### Meal Response Pattern
Shows average pre- and post-meal glucose response.

### Clinical Summary
Provides an interpretation of detected patterns.
